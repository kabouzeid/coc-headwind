# coc-headwind

Headwind integration, an opinionated class sorter for Tailwind CSS

## Install

`:CocInstall coc-headwind`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-headwind-keymap)`

## Lists

`:CocList demo_list`
